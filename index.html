/**
 * 获取配置文件
 *
 * @return 配置Props
 */
private static Properties getProperties() {
    // 读取配置文件
    Resource resource = new ClassPathResource("/config/application.properties");
    Properties props = new Properties();
    try {
        props = PropertiesLoaderUtils.loadProperties(resource);
    } catch (IOException e) {
        e.printStackTrace();
    }
    return props;
}

/**
 * 页面生成的文件名
 */
private static String getGeneratorViewPath(String viewOutputDir, TableInfo tableInfo, String suffixPath) {
    String name = StringUtils.firstToLowerCase(tableInfo.getEntityName());
    String path = viewOutputDir + "/" + name + "/" + name + suffixPath;
    File viewDir = new File(path).getParentFile();
    if (!viewDir.exists()) {
        viewDir.mkdirs();
    }
    return path;
}

linux入门

Share

Linux特征

windows：最通用操作系统，适用PC，应用广泛
mac：份额较低，安全性高
linux：适用于服务器终端，免费

常用版本

Ubuntu: 图形化界面
CentOs: 指令操作

命令 command

概念

在terminal中操作计算机的语句

命令格式

命令 [-options] [paramters]
parameter指定命令的参数，如mv rm的操作对象。形同宾语。
options指定命令的选项，如mv -i操作提示。形同定语、状语。

帮助指令：

command –help
或
man command
f/space 前滚一页
b 后退一页
enter 下一行
q 退出

常用命令

文件夹操作

command usage options
tree {path} 显示文件树 -d 只显示文件夹
ls {path} 显示所有文件 -lh 详细信息
-a 显示所有
cd {path} 打开某路径 
ls配合通配符使用：
    * 任意数量的字符
    ? 一个字符
    []其中一个字符
cd常用参数：
    -     最近使用的两个目录切换
    .     当前目录
    ..     上一级目录
    ~    home目录
    /    根目录
command usage options
rm {file} 删除 -r 删除文件夹
-f 强制删除无提示
mv {src} {targ} 移动、重命名 -r 移动文件夹
-i 覆盖提示
cp {src} {targ} 复制从src到targ -r 复制文件夹
-i 覆盖提示
touch {file} 创建文件、修改修改日期
mkdir 创建文件夹 -p 创建多个目录
cat {file} 查看 -b 显示非空行号
-n 所有行号
more {file} 分屏查看
grep {keyword} {file} 查找文本 {keyword}如有空格用””包裹 -n 显示行号
-v 取反
-i 忽略大小写
— grep可用正则表达式查找 ^a 查找以a开头
a$ 以a结尾
echo 控制台显示文本 配合重定向使用，可同时创建文件

命令增强

重定向：

> 覆盖某文件
>>追加进某文件

  • example: 

    echo hello world! > b
tree >> b
管道：

| 将一个命令的结果作为另一个命令的参数
常用管道：more grep

  • example: 

    ls -lha ~ | more
ls -lh | grep do
权限：

sudo

  • example: 

    sudo shutdown -r now

硬件相关

command usage options
shutdown {time} 关机，默认一分钟后关机 -r 重启
-c 取消
— time +10 十分钟
20:25 指定时间
now 立即
ifconfig 查看ip地址

  • 常用配合

    ifconfig | grep inet
